# Book Library Collection

A React app showcasing a personal collection of books, built with Vite and deployed on Vercel. This project uses reusable components to display book information in a clean, modern interface.

## Components

-   **`App.jsx`**: The main component that holds the book data and overall page structure (Header, Summary, Book List, Footer).
-   **`BookCard.jsx`**: A reusable component designed to display individual book information, including title, author, genre, year, and a brief description.

## Run Locally

To run this project on your local machine, follow these steps:

1.  Clone the repository:
    ```sh
    git clone https://github.com/YOUR_USERNAME/book-library-collection.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd book-library-collection
    ```
3.  Install the necessary dependencies:
    ```sh
    npm install
    ```
4.  Start the development server:
    ```sh
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.
